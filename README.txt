
____________________________________________SETUP_________________________________________________________________________________________				
1.) Start the Apache and MySQL modules using the XAMPP controller.
2.) Open the phpMyAdmin and create a datbase "ecommerce". 
3.) Import the sql files present in the "sql-tables" folder.
4.) Open the htdocs folder in the xampp folder. Copy paste the folder "ecommerce-website".
5.) Open the browser (chrome), type localhost/estore and you should see the index page of the website.
